Developer, DBA

The DBA is responsible for providing database support for the organization, including building infrastructure of database cluster, resolving database related incidents and requests, automating and managing scripts to audit changes and monitor system performance. Working closely with application team to review SQL statement, provide suggestion on SQL tuning, initiate and work with vendors to perform system upgrade.

Job Description

  • Manage the administration of databases in Oracle and MySQL environment -> Manage the administration of databases in PostgreSQL and MySQL Requirement part
  • Experience in Oracle and MySQL -> Experience in PostgreSQL and MySQL Willing to explore and research new features from open source projects
  • Responsible for performance tuning and query optimization using Performance Monitor, and other related monitoring tools.
  • Ability to detect and troubleshoot hardware resource shortage on CPU, memory, I/O, disk space, etc.
  • Setup clustering, database mirroring and replication as required.
  • Support application changes, implementations and upgrades.
  • Uses statistics to read and manage indexes for increased database performance
  • Implement operational automation wherever possible
  • Participate in off-hours support and rotating “on call” schedule

Core Requirements

  • Experience in Oracle and MySQL
  • Experience in optimizing common SQL performance issues
  • Experienced at performing all routine DBA activities including install, upgrade, patching and database management
  • Unix troubleshooting commands. Shell/awk/perl - one of these is mandatory
